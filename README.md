<h1>DoXTract</h1>

Very Fast Pure JS Text Extractor For Your Office Files.

<h2>Installation</h2>

```
npm i doxtract
```

<h2>Example</h2>

```js
const { extractText } = require('doxtract');
extractText('./file.docx').then(console.log).catch(console.error);
```
